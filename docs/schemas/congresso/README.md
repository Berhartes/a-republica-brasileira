# Schemas do Congresso

Este diretório contém os schemas relacionados ao domínio do Congresso Nacional.

## Estrutura

- `senadores.schema.ts` - Definição do schema para senadores
- `votacoes.schema.ts` - Definição do schema para votações
- `materias.schema.ts` - Definição do schema para matérias legislativas

## Utilização

Os schemas definidos aqui são utilizados para:

1. Validação de dados recebidos das APIs
2. Tipagem forte para as entidades do domínio
3. Documentação da estrutura de dados
