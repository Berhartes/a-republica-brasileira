# Schemas de Usuário

Este diretório contém os schemas relacionados ao domínio de usuários da aplicação.

## Estrutura

- `usuario.schema.ts` - Definição do schema para usuários
- `perfil.schema.ts` - Definição do schema para perfis de usuário
- `autenticacao.schema.ts` - Definição do schema para autenticação

## Utilização

Os schemas definidos aqui são utilizados para:

1. Validação de dados de usuário
2. Tipagem forte para as entidades do domínio
3. Documentação da estrutura de dados de usuário
